# 用户种子数据

本目录包含用于在系统中内置有特色用户的种子数据脚本。

## 功能说明

- 创建5个具有不同背景、兴趣和答题情况的用户
- 包含不同年龄、性别、职业和兴趣爱好
- 包含高级用户和免费用户示例
- 每个用户都有代表性的回答示例

## 如何运行

1. 确保已安装所有依赖：
   ```bash
   npm install
   ```

2. 确保已配置好环境变量（.env文件），包含MongoDB连接字符串等

3. 运行种子数据脚本：
   ```bash
   node seeds/seedUsers.js
   ```

## 注意事项

- 运行脚本前请确保MongoDB服务已启动
- 脚本会清空现有所有用户数据，谨慎使用
- 所有用户的初始密码均为：Password123!
- 可以根据需要修改`seedUsers.js`文件中的用户数据
